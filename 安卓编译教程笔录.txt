### 首先搭建安卓编译环境

1：自动安装JDK  
$ sudo apt-get install openjdk-6-jre-headless openjdk-6-jdk    

### 自动安装java6，以及设置JAVA环境。同理将6换成7也就是安装java7,但是目前谷歌指定了 2.3以上用JDK6，JDK7将会有很多你不知道的错误！如果说上诉命令失效那么你需要手动安装了（我是10月25安装的此命令有效）


#### 手动安装
卸载JDK
$ sudo apt-get purge openjdk-\* icedtea-\* icedtea6-\*

下载jdk1.6.0_45.bin
$ chmod 777 jdk-6u45-linux-x64.bin && ./jdk-6u45-linux-x64.bin                    #给.bin文件权限,运行.bin文件，得到jdk1.6.0_45文件夹                  

$ sudo cp -r ~/android/jdk1.6.0_45 /usr/lib/jvm

$ gedit .bashrc

# Java 环境
export JAVA_HOME=/usr/lib/jvm/jdk1.6.0_45
export PATH=$PATH:$JAVA_HOME/bin

$ source .bashrc                               #更新环境

$ sudo update-alternatives --install /usr/bin/java java /usr/lib/jvm/jdk1.6.0_45/bin/java 1062
$ sudo update-alternatives --install /usr/bin/javac javac /usr/lib/jvm/jdk1.6.0_45/bin/javac 1062

3: 安装android SDK
下载地址：http://developer.android.com/sdk/index.html

一般来说下载到"下载"文件夹

$ unzip 下载/adt-bundle-linux-x86_64-20130917.zip        #解压后得到adt-bundl……文件夹
$ mkdir android                        #创建android文件夹
$ cp -r 下载/adt-bundle-linux-x86_64-20130917/* android && rm -rf 下载/adt-bundle-linux-x86_64-20130917                  #将文件夹复制到android，删除文件夹！

### 加入SDK到环境变量

$ gedit .bashrc

### Android SDK环境变量
export PATH=${PATH}:~/android/sdk/tools
export PATH=${PATH}:~/android/sdk/platform-tools

### 添android SDK路径
PATH="$HOME/android/sdk/tools:$HOME/android/sdk/platform-tools:$PATH"

$ source .bashrc
